📋 Cómo usar:

  1. Abre desde Android con Chrome (no modo incógnito)
  2. Presiona "Iniciar Realidad Aumentada"
  3. Acepta permisos de cámara cuando el sistema lo pida
  4. Se abrirá Scene Viewer con la cámara real
  5. Coloca y escala el modelo con los gestos del teléfono

⚙️ Tecnologías utilizadas

Este proyecto usa tecnologías pensadas para que la experiencia funcione de forma rápida y directa desde el celular, sin instalar una aplicación extra. La idea es que el usuario solo abra el enlace, escanee el código QR y vea el modelo 3D en su entorno real con ayuda de la cámara.

  • WebAR: realidad aumentada desde el navegador, sin descargar app.
  • Android + cámara: el teléfono usa la cámara para superponer el modelo 3D en el entorno real.
  • Código QR: funciona como acceso inmediato al proyecto, llevando al usuario directamente a la página.
  • Modelo optimizado para móvil: el archivo 3D está preparado para cargar más rápido y verse mejor en celulares.

En resumen, la web muestra una vista 3D del personaje y, en dispositivos compatibles, permite abrir la experiencia de realidad aumentada para observar el modelo como si estuviera delante de la persona.
