\section{Riesgos, Indicadores y Anexos}
\subsection{Riesgos principales}
\begin{itemize}[leftmargin=*]
    \item Alcance excesivo dentro del Sprint.
    \item Dependencias t\'ecnicas en la visualizaci\'on 3D.
    \item Incidencias de autenticaci\'on o autorizaci\'on.
    \item Retraso en la integraci\'on de modelos 3D y personalizaci\'on permitida.
\end{itemize}

\subsection{Controles}
\begin{itemize}[leftmargin=*]
    \item Backlog ordenado por valor y riesgo.
    \item Sprint Goal corto y claro.
    \item Revisi\'on incremental con feedback real.
\end{itemize}

\subsection{Indicadores}
\begin{itemize}[leftmargin=*]
    \item Velocidad por Sprint.
    \item PBIs terminados vs. comprometidos.
    \item Defectos detectados en revisi\'on.
\end{itemize}

\subsection{Anexo de compilaci\'on}
Este paquete est\'a pensado para Overleaf. El archivo principal es \texttt{LEVANTAMIENTO_DOCUMENTACION.tex} y la bibliograf\'ia est\'a en \texttt{referencias\_scrum.bib}.

\subsection{Estado del proyecto}
La aplicaci\'on web se mantiene en desarrollo y los elementos no entregados se consideran pendientes o post-MVP seg\'un el backlog.
